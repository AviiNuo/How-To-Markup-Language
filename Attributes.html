HTML - Attributes

Attributes are used to provide additional information to HTML tags or elements.
The normal formating is <OpeningTag Attribute="info">
E.g. <p class="big"> (Don't forget the closing tag)
The main types of attributes are;

Class

The class attribute is used to connect several html tags and elements to any linked css files.
You relate back to classes in css files by using ".ClassName"

HTML                  CSS           Result


Id

The Id attribute is in many ways the same as the class attribute but it can only be used once, instead of multiple times. Also in the CSS file you use a # instead of a . (dot).

HTML            CSS         Result


Href

href stands for hypertext reference and is used to add a URL to tags such as anchor tags <a> and link tags.
On a anchor tag it would be used to redirect you when the anchor tag is clicked.
