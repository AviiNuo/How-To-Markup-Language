HTML - Tables

Tables are used to organise content into rows and columns. You use the <table> tag to define a table and you use the style attribute to change how the table looks. You use the <tr> tag to create rows inside the table and you use the <td> to add columns of data into the rows. The same can be done with the <th> tag except this makes the the column a heading column.
